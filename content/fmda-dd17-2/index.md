## Overview

The FMDA DD17.2 is the full-size Glock 17 variant of the Chairmanwon V2 series. Same improved screwless rear rail design as the DD19.2 with a longer slide and grip for full-size 17-round capacity. Multiple stipple patterns available.

## Parts Required

- DD17.2 Rail & Locking Block Kit
- G17 Gen 3 Lower Parts Kit
- G17 Gen 3 Complete Slide (barrel + recoil spring + upper parts)
- G17 Magazine

## Print Settings

| Setting      | Value                                        |
| ------------ | -------------------------------------------- |
| Nozzle       | 0.4mm                                        |
| Layer Height | 0.15mm                                       |
| Temp         | 225–230°C nozzle / 60°C bed                  |
| Infill       | 100%                                         |
| Supports     | On, tree preferred. Interface gap ≤ 0.15mm   |
| Cooling      | 20% normal, 50% bridges. Off first 10 layers |
| Orientation  | Rails down for best durability               |

## Notes

- Microtexture versions are for FDM with PLA only. Use no-microtexture for resin or nylon.
- Longer frame than DD19.2 — more prone to rail wear over time but higher capacity.
- Support X/Y distance: 1.3mm. Enable Z hop on retract.
