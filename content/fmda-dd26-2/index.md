## Overview

The FMDA DD26.2 is the subcompact Glock 26 variant of the Chairmanwon V2 series. Smallest frame in the lineup with a 10-round flush-fit capacity. Same improved screwless rear rail design. Tighter tolerances than the DD19.2 — requires more careful post-processing.

## Parts Required

- DD26.2 Rail & Locking Block Kit
- G26 Gen 3 Lower Parts Kit
- G26 Gen 3 Complete Slide (barrel + recoil spring + upper parts)
- G26 Magazine

## Print Settings

| Setting      | Value                                        |
| ------------ | -------------------------------------------- |
| Nozzle       | 0.4mm                                        |
| Layer Height | 0.15mm                                       |
| Temp         | 225–230°C nozzle / 60°C bed                  |
| Infill       | 100%                                         |
| Supports     | On, tree preferred. Interface gap ≤ 0.15mm   |
| Cooling      | 20% normal, 50% bridges. Off first 10 layers |
| Orientation  | Rails down for best durability               |

## Notes

- Subcompact grip has tighter mag well geometry — requires precise post-processing for reliable mag catch and feeding.
- Microtexture versions are for FDM with PLA only.
- Fewer community builds than the DD19.2 — less troubleshooting data available.
