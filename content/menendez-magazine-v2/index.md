## Overview

The Menendez Magazine v2.0 is a fully 3D printed Glock-pattern magazine available in multiple capacities. A complete redesign from v1.0 — solving issues with printability, lifespan, nosediving, and fitment consistency. Testers report 1,000+ dry cycles without mag body wear-out.

## Variants

| Variant       | Capacity  |
| ------------- | --------- |
| Mini Menendez | 10 rounds |
| Baby Menendez | 15 rounds |
| Menendez      | 17 rounds |
| FGC-9         | 25 rounds |
| Extendez      | 30 rounds |

## Baseplates

Three baseplate options to match different grip angles:

- **Standard** — flat baseplate
- **G17 Angle** — angled for Glock 17 grip
- **G19/G26 Angle** — angled for compact/subcompact grips

## Parts Per Magazine

Each magazine requires one printed set of:

- Magazine body (choose your capacity)
- Baseplate (choose your angle)
- Locking tab
- Follower
- Magazine spring (not printed — see assembly tutorial)

## Print Settings

| Setting      | Value                                                       |
| ------------ | ----------------------------------------------------------- |
| Nozzle       | 0.4mm (must be 0.4mm)                                       |
| Layer Height | 0.15mm (strongly recommended — affects feed lip geometry)   |
| Material     | eSUN PLA+                                                   |
| Temp         | 230°C nozzle / 60°C bed                                     |
| Infill       | 100%                                                        |
| Cooling      | Off by default. Auto-cooling 35% min / 60% max, 80% bridges |
| Wall Count   | 8 recommended                                               |
| Supports     | None on body, baseplate, locking tab. Optional on follower  |

## Notes

- No parts are compatible with v1.0 — completely new design.
- Wall count of 8+ helps reduce magazine swelling when loaded.
- Bed adhesion: glue stick or hairspray recommended.
